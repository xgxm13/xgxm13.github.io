node.js 的版本管理工具 Nvm 的安装及日常使用

  • 项目中经常遇到不同项目需要不同的 node.js 版本或者你本地 node 版本与客户现有系统的 node.js 版本相差较大等情况，我们肯定不可能卸载现有的再装客户需要的，这时 Nvm 就派上了用场。通过 Nvm，我们可以来回切换不同版本的 node.js，十分方便。

# Nvm 的安装

# mac 安装

  • 官方 md:https://github.com/nvm-sh/nvm/blob/master/README.md (多打开几次，比较慢)

  • 安装完成后关闭终端，重新打开终端输入 nvm 验证一下是否安装成功，当出现 Node Version Manager 时，说明已安装成功。如下图：（单击可放大）

# windows 安装

  • 下载 nvm-windows https://github.com/coreybutler/nvm-windows (多打开几次，比较慢)

# Nvm 的简单使用

  • nvm list
  • nvm use [version]
  • nvm install [version]
  • nvm uninstall [version] (不能卸载正在使用的版本，会报 nvm: Cannot uninstall currently-active node version, v6.17.1 (inferred from 6.17.1).)
  • nvm alias default [version]
